The intention is to provide access to all HP42S RPN commands or to provide an equivalent. Where it makes overwhelming sense, a more modern, time saving or Pythonic alternative is provided e.g. matrix access via indexing and slicing syntax - see the help file for information and tutorials.

Commands that operate on the stack, that use GTO, or which test and jump e.g. X<0? are not applicable in Python. And often it is simply that the symbols like ? in the RPN commands are not allowed in Python syntax, so something like FS? needs to be exposed as e.g. isFS().

Warning: The description column in the table below is the original "pure" HP42S command description, which will often refer to the stack. The stack is not used in Python so translate these descriptions accordingly. For example a references to stack x can often be substituted with a reference to a Python variable. E.g. SIN which acts on stack x is simply SIN(myvar) when accessed via Python.
